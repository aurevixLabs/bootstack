# Bootstack - Session 1 Minimal CLI

This folder is a learning-focused minimal version of Bootstack.

## What it includes

- Minimal interactive CLI prompt flow
- One template: React + Vite
- JavaScript / TypeScript choice
- Auto dependency installation

## Learning goals

1. Understand prompt collection with Inquirer
2. Understand command execution with child_process
3. Understand splitting logic into template modules
